# Phase 7.2 Research: Design System Overhaul

## Executive Summary

The current Omni Card landing page feels "childish" and unprofessional compared to fintech leaders like Bilt, Ramp, Mercury, and Stripe. This research identifies specific issues and proposes solutions based on competitive analysis.

---

## Competitive Analysis

### Bilt (bilt.com)
- **Theme**: Dark mode (`#010a13` deep navy)
- **Typography**: GT America VF (display) + Inter (body)
- **Colors**: Soft pastels on dark (`#97c6ea` blue, `#edd496` gold, `#fbdfc9` cream)
- **Feel**: Sophisticated, premium credit card company

### Ramp (ramp.com)
- **Theme**: Dark mode (`#010518` navy black)
- **Typography**: Bold hierarchical system with custom fonts
- **Colors**: High contrast, minimal accent colors
- **Feel**: Enterprise, professional, trustworthy

### Mercury (mercury.com)
- **Theme**: Multi-theme system (10+ palettes)
- **Typography**: Arcadia Display + Tiempos Fine (headlines), Arcadia (body)
- **Colors**: Blue primary (`#5266eb`), sophisticated frosted glass effects
- **Feel**: Premium banking, modern startup aesthetic

### Stripe (stripe.com)
- **Theme**: Light mode with generous whitespace
- **Typography**: Clean hierarchical system
- **Colors**: Minimal, data-driven
- **Feel**: Developer-focused, sophisticated, mature

## Current Omni Issues (Ranked by Impact)

### 1. CRITICAL: Storyset Illustrations
**Problem**: The Storyset "Online Transactions" and "College Campus" illustrations contain:
- Cartoon characters with exaggerated features
- Colorful decorative plants (#ba68c8 purple)
- Playful, consumer-app aesthetic
- Multiple unrelated visual elements

**Impact**: This single issue is likely the #1 reason the site feels "childish"

**Solution**:
- Option A: Revert to refined isometric SVG shapes (like original but polished)
- Option B: Remove illustrations, use clean data visualizations
- Option C: Commission minimal line art (expensive, time-consuming)

### 2. HIGH: Typography System
**Problem**:
- Inter is the most generic font in tech (used by everyone)
- No distinctive headline font
- Creates "template" feel

**Competitive comparison**:
| Site | Display Font | Body Font |
|------|-------------|-----------|
| Bilt | GT America | Inter |
| Mercury | Arcadia Display + Tiempos | Arcadia |
| Ramp | Custom display | System |
| Stripe | Custom | System |
| **Omni** | Inter | Inter |

**Solution**: Add a distinctive display font for headlines:
- Option A: **Geist** (Vercel's font - free, modern, clean)
- Option B: **Satoshi** (free, geometric, fintech-appropriate)
- Option C: **Cabinet Grotesk** (free, bold, distinctive)
- Keep Inter for body text

### 3. HIGH: Color Palette
**Problem**:
- Primary purple `oklch(0.50 0.22 270)` is highly saturated
- Creates vibrant, playful feel vs. sophisticated, trustworthy
- Hero gradient adds to "fun" aesthetic

**Competitive comparison**:
| Site | Primary Color | Saturation |
|------|--------------|------------|
| Bilt | Deep navy | Very low |
| Mercury | Blue #5266eb | Medium |
| Ramp | Navy/black | Minimal |
| Stripe | Purple accent | Low-medium |
| **Omni** | Bright purple | HIGH |

**Solution**:
- Reduce primary saturation: `oklch(0.45 0.15 270)` or `oklch(0.40 0.18 270)`
- Consider darker primary: `oklch(0.35 0.15 270)`
- Simplify hero gradient or use solid color
- Add neutral dark option for text emphasis

### 4. MEDIUM: Component Styling
**Problem**:
- Buttons are basic with simple hover states
- Cards lack depth layering
- Border radius (10px base) may be too rounded for fintech

**Competitive comparison**:
- Bilt: 8-17px radius, layered transparency
- Mercury: Comprehensive radius system, frosted glass
- Stripe: Subtle shadows, clean edges

**Solution**:
- Reduce base radius to 8px
- Add more sophisticated shadow layering
- Consider subtle borders on cards
- Add hover state depth (not just lift)

### 5. MEDIUM: Spacing & Density
**Problem**:
- Some sections feel cramped
- Could use more generous whitespace
- Metric cards could breathe more

**Solution**:
- Increase section padding (py-32 → py-40 on desktop)
- Add more gap in grid layouts
- Increase metric card padding

### 6. LOW: Hero Gradient
**Problem**:
- Animated gradient with multiple colors feels playful
- Premium sites use solid or subtle backgrounds

**Solution**:
- Option A: Remove animation, use subtle static gradient
- Option B: Single color with subtle texture
- Option C: Keep gradient but reduce saturation dramatically

## Recommended Design Direction

Based on research, here's the recommended approach for Omni:

### Visual Direction: "Clean Fintech"
- Light mode (like Stripe) but with more personality
- Distinctive headline typography
- Restrained color with strategic accent usage
- Clean geometric illustrations (not cartoons)
- Generous whitespace

### Specific Recommendations

1. **Remove Storyset illustrations immediately** - Replace with either:
   - Refined geometric isometric shapes
   - Or no illustrations (text + data visualization focus)

2. **Add display font**: Geist or Satoshi for headlines

3. **Adjust color palette**:
   ```css
   /* Current */
   --primary: oklch(0.50 0.22 270);  /* Too bright */

   /* Proposed */
   --primary: oklch(0.42 0.16 270);  /* Deeper, more sophisticated */
   --primary-light: oklch(0.55 0.14 270);  /* For hover states */
   ```

4. **Refine components**:
   - Reduce radius: 10px → 8px base
   - Add layered shadows
   - Improve button hover states

5. **Increase whitespace**:
   - Section padding: 128px → 160px desktop
   - Card padding: increase 50%
   - Grid gaps: increase 25%
