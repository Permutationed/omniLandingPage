# Plan 07.2-03: Color Palette Refinement - Reduce Saturation

## Objective

Reduce the saturation of the primary color palette to create a more sophisticated, mature fintech aesthetic. The current bright purple feels playful; premium fintech sites use deeper, more muted tones.

## Research Basis

**Current primary**: `oklch(0.50 0.22 270)` - High saturation (0.22)

**Competitive analysis**:
- Bilt: Near-zero saturation (navy/black)
- Mercury: Medium saturation blue (`#5266eb` ≈ oklch 0.52 0.18 265)
- Stripe: Low-medium saturation purple accents
- Ramp: Minimal saturation (dark theme)

**Proposed adjustment**:
- Reduce chroma from 0.22 to 0.15-0.18
- Slightly reduce lightness for more depth
- Keep hue 270 (purple/indigo family)

## Color Changes

### Primary Color
```css
/* Current */
--primary: oklch(0.50 0.22 270);

/* Proposed - Option A: Deeper & less saturated */
--primary: oklch(0.45 0.16 270);

/* Proposed - Option B: Even more refined */
--primary: oklch(0.42 0.15 270);
```

Recommendation: **Option A** (`oklch(0.45 0.16 270)`) - Noticeably more sophisticated while still having brand presence.

### Related Colors
Adjust these to maintain harmony:

```css
/* Current → Proposed */
--ring: oklch(0.50 0.22 270) → oklch(0.45 0.16 270);

/* Muted foreground - slightly less saturated */
--muted-foreground: oklch(0.50 0.02 270) → oklch(0.50 0.015 270);

/* Chart colors - reduce primary chart saturation */
--chart-1: oklch(0.50 0.22 270) → oklch(0.45 0.16 270);
```

### Hero Gradient
```css
/* Current - multiple bright colors */
background: linear-gradient(
  -45deg,
  oklch(0.94 0.04 280),
  oklch(0.96 0.03 250),
  oklch(0.98 0.02 270),
  oklch(0.95 0.035 200),
  oklch(0.97 0.025 290)
);

/* Proposed - more subtle, fewer colors */
background: linear-gradient(
  -45deg,
  oklch(0.96 0.02 270),
  oklch(0.98 0.015 265),
  oklch(0.97 0.018 275)
);
```

Or consider removing animation:
```css
.hero-gradient {
  background: oklch(0.98 0.015 270);
  animation: none;
}
```


The changes:
- Fewer gradient stops (3 instead of 5)
- Lower saturation on all stops (0.01-0.02 instead of 0.02-0.04)
- Smaller background-size (200% instead of 500%)
- Slower animation (30s instead of 25s)
